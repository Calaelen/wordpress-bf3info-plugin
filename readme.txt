== Description ==
Display your Battlefield 3 Player Statistics from bf3stats.com in a sidebar widget. Show/hide values via the settings (e.g. hide Origin username).

== Installation ==
1. Unpack the download-package
1. Upload all files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add a widget to your site via the design --> widget admin
1. Use the Settings -> Battlefield 3 Info Options Page to show/hide values
